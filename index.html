Conectar Google Sheets
ℹ O Odysseus lê diretamente da sua planilha pública no Google Sheets via URL. A planilha precisa estar publicada na web (Arquivo → Publicar na web → CSV).
1
Abra sua planilha no Google Sheets
Certifique-se de que a planilha tem o formato correto (Data, Descrição, Categoria, Valor)
2
Publique na web como CSV
Menu Arquivo → Publicar na web → selecione a aba → formato CSV → Publicar
3
Cole o link de publicação abaixo
O link termina com /pub?output=csv
